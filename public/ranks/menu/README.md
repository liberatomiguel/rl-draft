# Rank images — MENU art set

Drop one PNG per rank here (shown on the home screen). Exact filenames:

```
unranked.png
bronze.png
silver.png
gold.png
platinum.png
diamond.png
champion.png
grand-champion.png
supersonic-legend.png
```

Recommended: square images with transparent background, ≥160×160px.
While a file is missing, the app shows a styled CSS placeholder automatically
(component: `src/components/ui/RankBadge.tsx`, variant `menu`).
